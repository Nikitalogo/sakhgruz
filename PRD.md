# PRD — САХГРУЗ

## Что строим
Лендинг + админ-панель для транспортной компании
САХГРУЗ (Южно-Сахалинск).

## Для кого
Владелец управляет контентом сам через админку.
Клиенты находят контакты и связываются через мессенджеры.

## 7 ключевых функций
1. Публичный лендинг с услугами и ценами
2. Кнопки Telegram/WhatsApp — видны всегда
3. Галерея фото (заготовка, фото добавятся позже)
4. Отзывы из 2ГИС — управляются через админку
5. Вход в админку по паролю (только владелец)
6. Редактирование цен, услуг, контактов без программиста
7. Загрузка и удаление фото без программиста

## Что НЕ строим сейчас
- Форма заказа на сайте
- Несколько пользователей
- Онлайн-оплата
